# AI Product Content Generator

AI-powered Product Content Generator built with **FastAPI** and **Google Gemini AI**.

This project helps eCommerce admins generate high-quality product content automatically, including product titles, descriptions, SEO metadata, specifications, tags, and more.

---

## Features

- AI Product Title Generation
- AI Short Description
- AI Long Description
- SEO Meta Title
- SEO Meta Description
- SEO Keywords
- Product Specifications (JSON)
- Product Tags
- Prompt Engineering
- Structured JSON Output
- FastAPI REST APIs
- JWT Authentication
- SQLAlchemy ORM
- MySQL Database
